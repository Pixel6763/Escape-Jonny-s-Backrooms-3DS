# Escape-Jonny-s-Backrooms-3DS
By Pixel_bruh
Escape Jonny's Backrooms for the Nintendo 3DS is a 2D Backrooms game where you find you're way out of the Backrooms. While someone named Jonny chases you, and you have to survive and escape the maze. 4 difficulties: Easy, Medium, Hard, and Impossible! (This is based on a Roblox game I made a while ago named Escape Jonny's Backrooms. I had to make this from scratch because you cannot convert Roblox code (LUA) into 3DS code.)
